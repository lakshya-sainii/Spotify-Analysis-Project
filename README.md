# 🎵 Spotify Analytics Dashboard

A premium data analytics project built using **Python** and **Power BI** to analyze Spotify music trends, artist performance, song popularity, genres, and audio features through interactive visualizations and dashboards.

---

# 📌 Project Overview

This project focuses on extracting meaningful insights from Spotify track data using data analysis and business intelligence techniques.

The project includes:

* Data Cleaning & Exploration
* Data Visualization
* Statistical Insights
* Interactive Power BI Dashboard
* Music Trend Analysis

# 📈 Analysis Performed

## ✅ Top Artists Analysis

Identified the most active and popular artists in the dataset.

## ✅ Popularity Distribution

Analyzed how songs are distributed across popularity ranges.

## ✅ Danceability Analysis

Explored danceability trends among Spotify tracks.

## ✅ Energy vs Popularity

Compared audio energy levels with song popularity.

## ✅ Correlation Heatmap

Visualized relationships between numerical audio features.

## ✅ Songs Released Per Year

Observed music release trends over time.

## ✅ Genre Analysis

Identified the most dominant music genres.

## ✅ Explicit vs Non-Explicit Songs

Compared explicit and clean music distributions.

# 🎯 Key Insights

* Pop and Hip-Hop genres dominate Spotify streaming trends.
* High-energy and danceable songs generally perform better.
* Non-explicit tracks form the majority of songs in the dataset.
* Modern music releases have significantly increased over time.
