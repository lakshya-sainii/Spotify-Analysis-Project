# Spotify Data Analysis Project

## Overview

This project analyzes Spotify music data using Python and Power BI to discover trends in artist popularity, audio features, genres, and music streaming behavior.

---

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Power BI

---

## Analysis Performed

* Top Artists Analysis
* Popularity Distribution
* Danceability Analysis
* Energy vs Popularity
* Correlation Heatmap
* Songs Released Per Year
* Genre Analysis
* Explicit vs Non-Explicit Songs

---

## Power BI Dashboard

Created an interactive Spotify dashboard with:

* KPI Cards
* Genre Filters
* Popularity Insights
* Artist Analysis
* Trend Visualizations

---

## Key Insights

* Pop and Hip-Hop dominate Spotify datasets.
* Highly energetic songs tend to perform better.
* Non-explicit songs represent the majority of tracks.
* Modern music releases have increased significantly.
